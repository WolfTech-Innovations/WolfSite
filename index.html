About WolfOS

WolfOS is designed with modern workflows in mind. Our simple yet powerful configuration system allows you to install all the packages you need using a single config file after installation.

Simple Setup

Getting started with WolfOS is easy. Download the ISO, create a bootable drive, and follow our straightforward installation process. Once installed, you can customize your desktop environment to create your perfect workspace.

Effortless Updates

Staying current is simple with WolfOS. Just specify your packages and preferred kernel version in the config file, and you're ready to go.

For Everyone

From professional developers to everyday desktop users, WolfOS is designed to be approachable regardless of your experience level with Linux.
