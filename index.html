Overview

WolfOS is designed for developers, offering a simple configuration system that lets you install all the packages you need using a single config file after installation. Updates? It's easy! Specify the packages in the config, and you can even set your preferred kernel version.

Developers and Desktop Users alike can use it, no matter your expirenece in Linux, KDE makes it simple.

Installation

  1. Download the ISO from the Sourceforge Mirror.
  2. Create a bootable USB drive using your preferred tool (e.g., Rufus, Etcher).
  3. Boot your computer from the USB drive.
  4. Follow the on-screen instructions to install WolfOS.
