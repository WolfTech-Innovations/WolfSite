Everything you need

Touch-optimized interface

Phosh mobile desktop with on-screen keyboard and gesture navigation for seamless handheld gaming.

Steam + Multiple Launchers

Pre-installed Steam, Heroic (Epic/GOG), Lutris, and RetroArch. Proton for Windows games with full Vulkan support.

Gaming kernel

Liquorix kernel optimized for low latency with 1000Hz tick rate and performance tuning.

Vesktop with Vencord

Enhanced Discord client with Vencord built-in for better features, themes, and customization.

Privacy focused

No telemetry, no tracking, no data collection. Your gaming stays private.

Debian stable

Built on Debian Bookworm for rock-solid reliability and extensive software repositories.
