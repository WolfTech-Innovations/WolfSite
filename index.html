What's included

Phosh Desktop

GNOME-based mobile interface with touch gestures, adaptive apps, and responsive UI designed for handheld gaming.

Liquorix Kernel

High-performance kernel optimized for desktop and gaming. Low latency, smooth frame rates, better hardware support.

Steam + 32-bit support

Pre-installed with Steam, full 32-bit library support, Vulkan drivers, and Mesa for seamless Windows game compatibility.

Network Installer

Ultra-minimal installer at approximately 50MB. Downloads the latest base system during installation for always-fresh deployments.

Discord included

Discord pre-installed with desktop shortcut. Stay connected with voice chat and gaming communities.

Essential apps

Firefox, GParted, NetworkManager with WiFi support. Auto-login configured with Steam and Discord shortcuts on desktop.
