Install Tool Categories

Use these terminal commands to install specific categories of security tools from the official Kali repositories.

Terminal — Install Kali Tool Categories
$ sudo apt install kali-tools-information-gathering # Reconnaissance & OSINT tools
$ sudo apt install kali-tools-vulnerability-assessment # Vulnerability scanners & assessment
$ sudo apt install kali-tools-web-applications # Web application security testing
$ sudo apt install kali-tools-wireless # Wireless network security tools
$ sudo apt install kali-tools-exploitation # Exploitation frameworks & tools
$ sudo apt install kali-tools-forensics # Digital forensics & incident response
$ sudo apt install kali-tools-passwords # Password attacks & hash cracking
$ sudo apt install kali-tools-reverse-engineering # Reverse engineering & malware analysis
