What's Included

K9 Desktop Environment

Custom X11-based desktop with integrated window manager, compositor, blur effects, rounded windows, and system panel. Built from source during installation.

Minimal Xorg

Only essential X11 packages installed. No bloat, just what's needed for K9 to run smoothly with optimal performance.

Live Boot System

ISO boots directly into K9 desktop with overlay filesystem. All changes stored in RAM for a clean, fresh experience every boot.

Network Ready

NetworkManager included for WiFi and Ethernet support. Connect to your network and browse the web immediately.

System Tools

GParted for disk management, nano and vim for text editing, essential utilities for system administration.

Auto-login

Configured with automatic login and X11 startup. Boot straight into K9 desktop with no login prompts or delays.
